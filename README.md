# Pharma Use Case Project

This repository contains a comprehensive pharmaceutical use case implementation with the following structure:

## 📁 Directory Structure

- **use-case-document/** - Use case documentation and requirements
- **pbix-files/** - Power BI report files (.pbix)
- **dax-functions/** - DAX formulas and custom functions
- **achievements/** - Project milestones and achievements tracking

## 🎯 Project Overview

This project demonstrates a complete Power BI analytics solution for pharmaceutical business scenarios, including data modeling, DAX calculations, and visualization.

## 📝 Quick Start

1. Review use case documentation in `use-case-document/`
2. Explore Power BI files in `pbix-files/`
3. Check DAX functions in `dax-functions/`
4. Track progress in `achievements/`
